# ConectaPET - Sistema de Gestão de Doações

## Descrição do Projeto
Sistema de Gestão de Doações desenvolvido para a ONG parceira **Aconchego Felino** (Petrópolis, RJ). O sistema visa centralizar o cadastro de doadores, gerenciar o estoque de doações (como sachês de alimentação e medicamentos) e emitir relatórios mensais, eliminando o uso de planilhas manuais.

## Estrutura de Pastas
* `/docs`: Documento de Visão em PDF, diagramas MER e wireframes.
* `/backend`: API REST desenvolvida em Node.js com Express.
* `/frontend`: Aplicação web desenvolvida em React.
* `/database`: Scripts SQL de criação de tabelas e seeds iniciais.

## Instruções de Instalação
1. Clone este repositório para sua máquina local.
2. Copie o arquivo `.env.example` e renomeie para `.env`, preenchendo as variáveis de ambiente necessárias.
3. Acesse a pasta `/backend`, execute `npm install` e depois inicie com `npm start`.
4. Acesse a pasta `/frontend`, execute `npm install` e inicie a interface com `npm start`.
